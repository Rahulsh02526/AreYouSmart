📋 How to use this tool
1. Click Load JSON to open a question file (e.g. ipl.json or worldcup.json)
2. View and edit questions in All Questions
3. Add new questions using Add Question
4. Review outdatable questions (those with changing facts) in Needs Review
5. Click Download JSON to save the updated file
6. Upload the file to your GitHub repo — Cloudflare auto-deploys in 30 seconds
⚠️ Outdatable Questions
Questions marked outdatable: true have answers that may change (e.g. "Most IPL titles"). The review_after date tells you when to check. After each IPL season, filter by "Needs Review" and update in bulk.
